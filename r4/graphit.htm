GraphIt
presents a line graph, with 1 or 4 quadrants

Categories: 

 line graph

Usage: 

 GRAPHIT GraphPropertyFileName

Arguments:

If the file name argument contains spaces it must be surrounded by double-quotes. 

 GraphPropertyFileName -- file that contains graph properties

Redirected input file: 

 N/A

Redirected output file: 

 N/A

Returns: 

  0 when graph was presented successfully
 99 incorrect usage

Description:

The GraphIt program presents a line graph, with 1 or 4 quadrants. The graph automatically adjusts when you resize the window with the mouse (i.e. by dragging the lower right corner). The program is controlled by a series of properties that are specified in the GraphPropertyFileName. Each property line has a property name followed by an equal sign and then the corresponding property value. A pound sign character at the beginning of a line causes the line to be ignored.

quadrants property

The quadrants property defines the number of quadrants to prepare: 1 or 4. (The default is 1 quadrant)

X and Y axes properties:

The following properties affect the preparation of the X and Y axes.

xOrigin
Defines the base X value

    For example if the X origin is 1990, and a point's X value is 1999, then the point is drawn at X position 9.

yOrigin
Defines the base Y value (default: 100)

    For example if the Y origin is 20, and a point's Y value is 47, then the point is drawn at Y position 27.

xLimit
Defines the highest X value (default: 100)

yLimit
Defines the highest Y value (default: 100)

xTicks
Defines the position of X tick marks

yTicks
Defines the position of Y tick marks

xTickCaptions
Defines the captions to display under the X tick marks

Color properties:

The link below describes how color parameters are specified. A table of supported color names is also displayed.

 Color table

The GRAPHIT program supports the following color properties:

bgColor
Defines the color of the area outside of the graph.

fgColor
Defines the color of the graphed line, when a single line is presented. When multiple lines are displayed, the colorSet property defines the colors of each line.

gridColor
Defines the color of graph grid lines.

insideColor
Defines the color that appears inside the graph.

textColor
Defines the color for captions, etc..

Multiple line properties:

The following properties affect the display of multiple lines.

lineCount
Defines the number of lines that are graphed

colorSet
Defines the color for each line

imageSet
Defines the image(s) to display at each line vertex

pointSet1, pointSet2, etc
Defines the points associated with each line

Text font properties:

The following properties affect the display of captions.

fontName
The name of the text font. The default font is SansSerif. Supported font names are:

  • Dialog
  • DialogInput
  • Monospaced
  • SansSerif
  • Serif

fontSize
This is the point size of the text font; i.e. 12.

fontStyle
This is the style of text presentation. Supported styles are:

  • Dialog
  • DialogInput
  • Monospaced
  • SansSerif
  • Serif

Other properties, 4-quadrant graphs:

The following additional properties are supported for graphs with 4 quadrants.

xCaption
Defines the caption that appears under the X axis.

yCaption
Defines the caption that appears vertically to the left of the Y axis.

Example 1: single line graph, with image at each vertex 

 GraphIt "Graph1.props"

# Graph1.props
quadrants=1
bgColor=gainsboro
fgColor=#ffff00
gridColor=gainsboro
insideColor=#ff0000
imageSet=person.gif
pointSet=0,1;10,5;20,14;30,22;40,35;50,50;60,70;
xOrigin=0
yOrigin=0
xLimit=70
yLimit=80
xTicks=0;10;20;30;40;50;60
xTickCaptions=1999;2000;2001;2002;2003;2004;2006
yTicks=10;20;30;40;50;60;70
